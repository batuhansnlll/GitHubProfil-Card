# Profil Kartı 🪪

Fareyle hareket ettirince 3D dönen, animasyonlu kişisel profil tanıtım kartı.

---

## ✨ Özellikler

- 🖱️ Fareyle 3D eğilme efekti
- ✨ Arka planda yüzen partiküller
- 💜 Avatar nabız ışıltı animasyonu
- 🏷️ Teknoloji rozetleri
- 🔢 Sayaç animasyonu (proje & commit)
- 📧 GitHub ve e-posta linkleri

---

## 🚀 Kullanım

1. `profil-karti.html` dosyasını indirin
2. Tarayıcıda açın
3. Fareyle kartın üzerine gelin
